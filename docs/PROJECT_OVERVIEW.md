# PROJECT OVERVIEW

## Название проекта
**Путь преодоления** - одностраничный сайт (Showcase) для анонимной помощи людям с зависимостью от наркотиков и алкоголя в Ивановской области.

## Миссия
Показать, что выход есть, дать надежду и безопасную точку входа в восстановление через понятный интерфейс и прямой контакт с горячей линией.

## Целевая аудитория
- Люди в кризисной ситуации, которым нужна срочная поддержка.
- Родственники и близкие, ищущие помощь для зависимого человека.
- Люди, которые сомневаются и боятся первого шага.

## Основные продуктовые цели MVP
1. Сделать номер горячей линии главным элементом интерфейса.
2. Объяснить путь выздоровления простым и поддерживающим языком.
3. Сформировать доверие через визуальный стиль (чистота, свет, безопасность, забота).
4. Обеспечить идеальный мобильный опыт (one-tap звонок и быстрый доступ к контактам).

## Продуктовые принципы
- **Безопасность:** никакого перегруза, никаких тревожных визуальных паттернов.
- **Ясность:** пользователь за 3-5 секунд понимает, куда нажать и как получить помощь.
- **Поддержка:** текст и визуал дают ощущение принятия, а не давления.
- **Анонимность:** акцент на конфиденциальности в каждом ключевом блоке.

## MVP-блоки
- Hero-секция с основным призывом к звонку.
- Блок "Поддержка и ценности" с анимированными карточками.
- Блок "Этапы выздоровления" с интерактивной шкалой прогресса.
- Блок "Наш центр" с легким glassmorphism-эффектом.
- Footer с крупным номером, адресом и ссылками на внешние площадки.

## Критерии успеха MVP
- Пользователь с мобильного может позвонить в 1 касание из Hero и Footer.
- Контент читается без усилий, CTA виден сразу на первом экране.
- Анимации плавные и ненавязчивые, не мешают восприятию.
- Lighthouse (mobile) >= 90 по Performance, Accessibility, Best Practices.
